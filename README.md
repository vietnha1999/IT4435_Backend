# How to run/build

### Install packages
```ruby
$ npm install
```
### Set up database
```ruby
$ Install xampp
$ Start Apache and MySQL modules
$ Add new account in phpmyadmin
```
### Setup environment
```ruby
$ Create a .env file, set up like:
PORT=8080
DB_HOST=localhost
DB_PORT=3306
DB_NAME=test
DB_USERNAME=test
DB_PASSWORD=123456
SECRECT_KEY=1a2b3c4d
```
### Run
```ruby
$ npm start
```
### Build
```ruby
$ npm run build
```
